My projects

Web site in English

This website project in English was a first-year end project at the IUT Informatique in Lannion, we had a subject, a photographer’s site, and we had to create in a few weeks a portal where interested people could see what the photographer proposed with some photo to illustrate in a carousel. And with a little more time and knowledge, we had the idea to set up a system to book an appointment with the photographer on a calendar, a way to book a photo shoot in the outdoor or in the studio.

Traductor UML To Java

The concept of this project is much simpler to explain than to implement, it is about creating an application where the user can create a UML diagram and the software automatically translates it. The most difficult was to set up the translation, the solution we found was to translate the code and display it in a different tab so the user can copy the code and put it in his compiler.
